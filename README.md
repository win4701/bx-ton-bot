# BX TON Telegram Bot

Production-ready Telegram bot for **TON + Jetton (BX)** built with **Node.js**.  
Includes wallet connection, balance, claim/airdrop, withdrawals, anti-fraud limits, and an optional Telegram **Mini App**.

---

## Features

- Telegram Bot (Polling)
- TON wallet connection (Deep Link)
- Jetton (BX) balance, claim, withdraw
- Anti-fraud protections (daily limits, one-time claim)
- PostgreSQL database
- Optional Telegram Mini App (WebApp)
- Ready for Render / VPS deployment
